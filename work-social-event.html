ФУТБОЛ

Компания оплачивает аренду футбольного поля для проведения любительских игр в мини-футбол. Обеспечиваются душевые кабины для послеигровой гигиены, помещение для переодевания.

Формируются команды по 5 человек. Игра "на вылет".

Общее время 1 мероприятия: 2 часа.

Для участия необходимо иметь: 1. Сменную форму 2. Полотенце 3. Гигиенические принадлежности для душа 4. Питьевая вода - по необходимости

Внимание: ценные вещи, телефоны и деньги в помещении для переодевания рекомендуется не оставлять.
